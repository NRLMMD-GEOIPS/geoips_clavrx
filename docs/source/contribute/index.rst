Contribute
***********

We encourage active participation by the user and developer community,
and we welcome contributions to the geoips_clavrx plugin.

Contributions generally follow the process of:

1. Submit a GitHub Issue for a bug fix or feature request.
2. Branch off the main branch to make the required changes.
3. Create a GitHub Pull Request for your changes to be reviewed.
4. Ensure all required tests pass prior to PR approval.
5. Merge your changes into the main code base!
