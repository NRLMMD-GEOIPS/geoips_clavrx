geoips_clavrx
=============

The geoips_clavrx package is a GeoIPS-compatible plugin that provides the capability for
reading and plotting data files produced from the Clouds for AVHRR Extended (CLAVR-x)
package. It also contains wrappers for installing and running the main CLAVR-x package
prior to processing CLAVR-x outputs through GeoIPS.
